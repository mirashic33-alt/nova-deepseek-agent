# Как применить тему Nova Dark

Сейчас стоит персиковая тема («Персиковый чай»). Рядом лежит `theme_nova.py` с тёмной космической темой.

## Чтобы переключиться:

1. Закрой агента если запущен
2. Переименуй `theme.py` → `theme_peach.py` (сохраняем старую)
3. Переименуй `theme_nova.py` → `theme.py`
4. Запусти агента снова

## Чтобы вернуть персиковую:

1. Переименуй `theme.py` → `theme_nova.py`
2. Переименуй `theme_peach.py` → `theme.py`
3. Перезапусти
